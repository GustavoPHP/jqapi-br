.jquery

Categorias: Internals | Properties > Properties of jQuery Object Instances

jquery Returns: String

Descrição: Uma string contendo o número da versão do jQuery.

  • adicionado na versão: 1.0jquery

A propriedade .jquery é associada ao jQuery prototype, associado pelo alias $.fn. Na string terá o número da versão do jQuery, como "1.5.0" ou "1.4.4".

Exemplos:

Exemplo: Verifica se o objeto é um objeto jQuery 


var a = { what: "Um objeto JS regular" },
    b = $('body');
     
if ( a.jquery ) { // falsy, since it's undefined
    alert(' a is a jQuery object! ');    
}

if ( b.jquery ) { // truthy, since it's a string
    alert(' b is a jQuery object! ');
}

Exemplo: Recebe a versão corrente do jQuery que está incluído na página 


alert( 'You are running jQuery version: ' + $.fn.jquery );

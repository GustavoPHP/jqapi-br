.hasClass()

Categories: Attributes | Manipulation > Class Attribute | CSS

.hasClass( className ) Retorna: Boolean

Descrição: Este método destina-se a inspecionar se um elemento possui o atributo class com um determinado valor.

  • criado na versão: 1.2 .hasClass( className )

    className O valor da classe a se inspecionar.

É válido definir mais de um valor para o atributo class de um elemento. A sintaxe HTML prevê, nestes casos, que os valores do atributo class sejam separados por um espaço, com mostrado a seguir:

<div id="mydiv" class="foo bar"></div>

O método .hasClass() retorna true se o valor da classe estiver presente no atributo class, mesmo que existam outros valores naquele atributo. Por exemplo: Para a marcação HTML mostrada anteriormente os dois códigos a seguir retornam true:

$('#mydiv').hasClass('foo')

e:

$('#mydiv').hasClass('bar')

O código a seguir retorna false:

$('#mydiv').hasClass('quux')

Exemplo:

Procurando o parágrafo que tem o valor 'selected' no atributo class.

<!DOCTYPE html>
<html>
<head>
  <style>
  p { margin: 8px; font-size:16px; }
  .selected { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
  <p>Este é o primeiro parágrafo e tem a cor preta.</p>
  <p class="selected">Este é o segundo parágrafo e tem a cor vermelha.</p>

  <div id="result1">O primeiro parágrafo tem o valor selected no atributo class: </div>
  <div id="result2">O segundo parágrafo tem o valor selected no atributo class: </div>
  <div id="result3">Pelo menos um parágrafo tem o valor selected no atributo class: </div>
<script>
$("div#result1").append($("p:first").hasClass("selected").toString());
$("div#result2").append($("p:last").hasClass("selected").toString());
$("div#result3").append($("p").hasClass("selected").toString());
</script>

</body>
</html>

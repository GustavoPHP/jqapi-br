event.relatedTarget

Categoria: Events > Event Object

event.relatedTarget Retorna: Element

Descrição: Outro elemento DOM envolvido no evento, se tiver algum.

  • adicionado na versão: 1.1.4event.relatedTarget

Para mouseout, indica qual o elemento entrou; Para mousein, indica qual elemento que saiu.

Exemplo:

Nas âncoras do mouseout, exibe um 'alert' do tipo do elemento que entrou.

$("a").mouseout(function(event) {
  alert(event.relatedTarget.nodeName); // "DIV"
});

.select()

Categories: Events > Form Events | Forms

.select( handler(eventObject) ) Retorna: jQuery

Description: Atrela um manipulador de evento ao evento JavaScript "select", ou dispara o evento em um elemento.

  • criado na versão: 1.0 .select( handler(eventObject) )

    handler(eventObject) Função a ser executada toda vez que o evento ocorre.

  • criado na versão: 1.4.3 .select( [ eventData ], handler(eventObject) )

    eventData Mapa de dados a serem passados ao manipulador do evento.

    handler(eventObject) Função a ser executada toda vez que o evento ocorre.

  • criado na versão: 1.0 .select()

Nos dois primeiros casos mostrados anteriormente este método é uma forma abreviada para o método .bind('select', handler) e na terceiro para o método .trigger('select').

O evento select ocorre em um elemento quando o usuário faz uma seleção no elemento. Este evento aplica-se a um conjunto limitado de elementos, tais como elementos de formulário <input type="text"> e <textarea>.

Considere a seguinte marcação HTML:

<form>
  <input id="target" type="text" value="Ei você" />
</form>
<div id="other">
  Dispara o evento
</div>

O evendo pode ser atrelado ao campo de texto:

$('#target').select(function() {
  alert('Chamado o manipulador do evento .select()');
});

Ao ser selecionado qualquer trecho de texto o alerta é mostrado. Não basta apenas apontar a inserção no texto, é preciso fazer uma seleção para disparar o evento. Para disparar o evento manualmente basta aplicar .select() sem argumentos:

$('#other').click(function() {
  $('#target').select();
});

Segundo este código, clicando o elemento para disparar o evento o alerta é mostrado:

Chamado o manipulador do evento .select()

Além disto a ação select será disparada, causando a seleção de todo o texto.

O método para inspecionar o texto selecionado varia de acordo com o navegador. Existem vários plugins jQuery que tornam o método consistente pelos navegadores e plataformas.

Exemplo: Faça alguma coisa quando o texto for selecionado: 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { color:blue; }
  div { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>
    Selecione textos nos campos.
  </p>
  <input type="text" value="Aqui um texto para teste" />
  <input type="text" value="e aqui mais texto para teste" />
  <div></div>
<script>
    $(":input").select( function () { 
      $("div").text("Houve uma seleção").show().fadeOut(1000); 
    });
</script>
</body>
</html>

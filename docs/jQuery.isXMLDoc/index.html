jQuery.isXMLDoc()

Categorias: Utilities

jQuery.isXMLDoc( node ) Retorna: Boolean

Descrição: Verifica se um nó do DOM está dentro de um documento XML (ou se é um documento XML).

  • versão adicionada: 1.1.4jQuery.isXMLDoc( node )

    nodeO nó do dom que será verificado se é ou não um documento XML.

Exemplo:

Check an object to see if it's in an XML document.

jQuery.isXMLDoc(document) // false
jQuery.isXMLDoc(document.body) // false

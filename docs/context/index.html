.context

Categorias: Internals | Properties > Properties of jQuery Object Instances

context Retorna: Element

Descrição: O nó contextual do DOM passado ao jQuery; Se nada for passado, o contexto provavelmente será o document

  • adicionado na versão: 1.3context

O método .live() para vincular event handlers utiliza esta propriedade para determinar o elemento root que irá utilizar para as delegações necessárias. Plug-ins que executam tarefas semelhantes também pode utilizá-lo.

O valor desta propriedade é tipicamente igual ao document, The value of this property is typically equal to document, pois este é o contexto padrão para objetos jQuery, se nada for passado. O contexto pode ser diferente se, por exemplo, o objeto foi criado através de pesquisa dentro de um <iframe> ou documento XML.

Exemplo:

Determina exatamente o contexto utilizado.

<!DOCTYPE html>
<html>
<head>
  <style>
  body { cursor:pointer; }
  div { width:50px; height:30px; margin:5px; float:left;
        background:green; }
  span { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Context:<ul></ul>
<script>$("ul")
  .append("<li>" + $("ul").context + "</li>")
  .append("<li>" + $("ul", document.body).context.nodeName + "</li>");

</script>

</body>
</html>

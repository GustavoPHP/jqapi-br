.change()

Categories: Events > Form Events | Forms

.change( handler(eventObject) ) Retorna: jQuery

Descrição: Atrela um manipulador de evento ao evento JavaScript "change", ou dispara o evento em um elemento.

  • criado na versão: 1.0 .change( handler(eventObject) )

    handler(eventObject) Função a ser executada toda vez que o evento ocorre.

  • criado na versão: 1.4.3 .change( [ eventData ], handler(eventObject) )

    eventData Mapa de dados a serem passados ao manipulador do evento.

    handler(eventObject) Função a ser executada toda vez que o evento ocorre.

  • criado na versão: 1.0 .change()

Nos dois primeiros casos mostrados anteriormente este método é uma forma abreviada para o método .bind('change', handler) e na terceiro para o método .trigger('change').

O evento change ocorre em um elemento quando seu vapr muda. Este evento ocorre nos elementos <input>, <textarea> e <select>. Para select boxes, checkboxes e radio buttons o evento é disparado quando o usuário faz uma seleção. Para os demais elementos o evento dispara somente quando o elemento perde o foco.

Considere a seguinte marcação HTML:

<form>
  <input class="target" type="text" value="Field 1" />
  <select class="target">
    <option value="option1" selected="selected">Opção 1</option>
    <option value="option2">Opção 2</option>
  </select>
</form>
<div id="other">
  Dispara o evento
</div>

O evento pode ser atrelado ao campo de texto e à caixa de seleção.

$('.target').change(function() {
  alert('Chamado o manipulador do evento .change()');
});

Quando a segunda opção é selecionado o alerta é mostrado. É mostrado também quando se altera o campo de texto e tira-se o foco dele. Para disparar o evento manualmente use .change() sem parâmetros:

$('#other').click(function() {
  $('.target').change();
});

Segundo este código disparando o evento o alerta é mostrado. O alerta será mostrado duas vezes pois o evento change foi atrelado aos dois campos do formulário.

A partir da jQuery 1.4 o evento change se propaga pelo efeito bolha (bubble) no Internet Explorer e é consistente nos navegadores modernos.

Exemplos:

Exemplo: Atrela o evento change ao elemento select e cada vez que uma opção é escolhida escreve-a em um div. 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { color:red; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <select name="sweets" multiple="multiple">
    <option>Chocolate</option>
    <option selected="selected">Candy</option>
    <option>Taffy</option>
    <option selected="selected">Caramel</option>
    <option>Fudge</option>
    <option>Cookie</option>
  </select>
  <div></div>
<script>
    $("select").change(function () {
          var str = "";
          $("select option:selected").each(function () {
                str += $(this).text() + " ";
              });
          $("div").text(str);
        })
        .change();
</script>
</body>
</html>

.empty()

Categorias: Manipulation > DOM Removal

.empty() Retorna: jQuery

Descrição: Remove todos os nós filhos contidos no conjunto de elementos do DOM.

  • versão que foi adicionada: 1.0.empty()

Este método não remove apenas um filho (e outros descendentes) elementos, mas também todo o texto dentro do conjunto de elementos combinados. Isso porque, de acordo com a especificação do DOM, qualquer cadeia de texto dentro de um elemento é considerado um nó filho deste elemento. Considere o seguinte código HTML:

<div class="container">
  <div class="hello">Hello</div>
  <div class="goodbye">Goodbye</div>
</div>

Nós podemos selecionar qualquer elemento para remover:

$('.hello').empty();

Isto irá resultar na estrutura DOM com o texto Hello excluído:

<div class="container">
  <div class="hello"></div>
  <div class="goodbye">Goodbye</div>
</div>

Se tivéssemos qualquer número de elementos aninhados dentro do <div class="hello">, eles seriam removidos também.

Para evitar desperdício de memória, o jQuery remove outros conteúdos, como dados e manipuladores de eventos a partir dos elementos filhos antes de remover os próprios elementos.

Exemplo:

Remove todos os nós filhos (incluindo os textos) de todos os parágrafos.

<!DOCTYPE html>
<html>
<head>
  <style>
  p { background:yellow; }
</style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>
  Hello, <span>Person</span> <a href="javascript:;">and person</a>
</p>

<button>Call empty() on above paragraph</button>
<script>
  $("button").click(function () {
    $("p").empty();
  });
</script>

</body>
</html>

.removeClass()

Categories: Attributes | Manipulation > Class Attribute | CSS

.removeClass( [ className ] ) Retorna: jQuery

Descrição: Este método remove uma ou mais classes de cada um dos elementos alvo de um seletor jQuery.

  • criado na versão: 1.0.removeClass( [ className ] )

    classNameUm ou mais nomes a serem removidos do atributo classe de cada elemento alvo de um seletor jQuery.

  • criado na versão: 1.4.removeClass( function(index, class) )

    function(index, class)Define uma função retornando um ou mais nomes de classe, separados por vírgula, para serem removidos. Os argumentos desta função são: o índice que define a posição do elemento no conjunto alvo da seleção e o valor da atual classe.

Se um valor de classe for incluído como parâmetro deste método, somente aquele valor será removido. Não havendo parâmetro todos os valores do atributo class serão removidos.

É possível remover mais de um valor de classe ao mesmo tempo separando-os por vírgula como mostrado a seguir:

$('p').removeClass('myClass yourClass')

Este método é usado com frequência em conjunto com o método .addClass() com a finalidade de proceder uma troca do valor do atributo class de um elemento conforme mostrado a seguir:

$('p').removeClass('myClass noClass').addClass('yourClass');

Neste exemplo os valores myClass e noClass são removidos de todos os parágrafos e o valor yourClass é adicionado.

Para fazer a troca de classes podemos ainda usar o método .attr('class', 'newClass').

A partir da versão 1.4 da biblioteca jQuery o método .removeClass() pode receber uma função como argumento.

$('li:last').removeClass(function() {
          return $(this).prev().attr('class');
        });

Neste exemplo é removido o atributo classe do penúltimo elemento <li>.

Exemplos:

Exemplo: Remove o valor 'blue' do atributo class do elemento alvo do seletor jQuery. 

<!DOCTYPE html>
<html>
<head>
  <style>

  p { margin: 4px; font-size:16px; font-weight:bolder; }
  .blue { color:blue; }
  .under { text-decoration:underline; }
  .highlight { background:yellow; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p class="blue under">Hello</p>
  <p class="blue under highlight">and</p>
  <p class="blue under">then</p>

  <p class="blue under">Goodbye</p>
<script>$("p:even").removeClass("blue");</script>

</body>
</html>

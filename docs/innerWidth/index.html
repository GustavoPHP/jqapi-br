.innerWidth()

Categories: CSS | Dimensions | Manipulation > Style Properties

.innerWidth() Retorna: Integer

Descrição: Este método retorna o valor da propriedade CSS width definida para o primeiro elemento do conjunto de elementos alvo do seletor jQuery. No valor retornado está incluído o valor de padding (se existente) mas não o de border.

  • criado na versão: 1.2.6 .innerWidth()

Este método retorna o valor da largura do elemento em pixels, incluíndo padding esquerdo e direito.

Este método não se aplica aos objetos window e document, para estes objetos use o método .width().

Example:

Retorna innerWidth do parágrafo.

<!DOCTYPE html>
<html>
<head>
  <style>p { margin:10px;padding:5px;border:2px solid #666; } </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Hello</p><p></p>
<script>var p = $("p:first");
$("p:last").text( "innerWidth:" + p.innerWidth() );</script>

</body>
</html>

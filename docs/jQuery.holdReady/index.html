jQuery.holdReady()

Categories: Core

jQuery.holdReady( hold ) Retorna: Boolean

Descrição: Segura ou libera a execução do evento do jQuery ready

  • adicionado na versão: 1.6jQuery.holdReady( hold )

    holdIndica se o método ready hold está sendo solicitado ou liberado.

O método $.holdReady() habilita que o chamado do evento jQuery ready atrase. Esse recurso avançado seria normalmente usado por scripts de carregamento dinâmico que desejem carregar JavaScripts adicionais como, tais como plugins do jQuery antes do evento ready esteja completo, mesmo que o DOM esteja ready(carregado). Este método pode ser chamado no início do documento, como imediatamente após a tag do jQuery script. Chamando este método após o evento ready já disparado não terá efeito nenhum.

Para atrasar o evento ready, primeiro execute $.holdReady(true). Quando o evento ready deveria ser liberado para executar, execute $.holdReady(false). Perceba que multiplos atrasos podem ser colocados no evento ready, um de cada vez $.holdReady(true) execute. O evento ready não irá disparar até que todos os holds(atrasos) tenham sido liberados com um correspondente $.holdReady(false) e a condição normal do 'document ready' seja encontrada. (Acesse ready para mais informações.)

Exemplo:

Atrasar o evento ready até que um plugin customizado seja carregado.


$.holdReady(true);
$.getScript("myplugin.js", function() {
     $.holdReady(false);
});

event.preventDefault()

Categoria: Events > Event Object

event.preventDefault() Retorna: undefined

Descrição: Se este método é chamado, a ação padrão do evento não é disparada.

  • adicionado na versão: 1.0event.preventDefault()

Por exemplo, clicando nas âncoras não irá colocar o browser em uma nova URL. Nós podemos utilizar event.isDefaultPrevented() para determinar se este método foi chamado por algum manipulador que foi disparado neste evento.

Exemplo:

Cancela a ação padrão (navegação) do click.

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<a href="http://jquery.com">default click action is prevented</a>
<div id="log"></div>

<script>
$("a").click(function(event) {
  event.preventDefault();
  $('<div/>')
    .append('default ' + event.type + ' prevented')
    .appendTo('#log');
});
</script>

</body>
</html>

.offset() Retorna: Object

Description: Este método retorna um objeto contendo os valores das coordenadas (relativas ao documento) do primeiro elemento do conjunto de elementos alvo do seletor jQuery.

  • criado na versão: 1.2 .offset()

O método .offset() permite inspecionar a posição de um elemento em relação ao documento. Difere do método .position() porque com este inspeciona-se a posição de elemento em relação ao seu elemento-pai. Este método é útil para posicionar um elemento sobre outro quando se pretende realizar manipulação global com ele (em particular para implementação de drag-and-drop).

O método .offset() retorna um objeto contendo as propriedades CSS top e left.

Nota: Não existe um método jQuery para inspecionar as coordendas de elementos estilizados como hidden e. Este método não considera para cálculo de coordenadas os valores de borders, margins ou padding definidos para o elemento body.

Exemplos:

Exemplo: Inspecionar as coordenadas do segundo parágrafo: 

<!DOCTYPE html>
<html>
<head>
  <style>
p { margin-left:10px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Olá</p><p>segundo parágrafo</p>
<script>var p = $("p:last");
var offset = p.offset();
p.html( "left: " + offset.left + ", top: " + offset.top );</script>

</body>
</html>

.offset( coordinates ) Retorna: jQuery

Descrição: Define as coordenadas (relativas ao documento) para cada um dos elementos alvo do seletor jQuery.

  • criado na versão: 1.4 .offset( coordinates )

    coordinates Objeto contendo as propriedades top e left (números inteiros) definindo as novas coordenadas dos elementos.

  • criado na versão: 1.4 .offset( function(index, coords) )

    function(index, coords)Define uma função retornando o valor das novas coordenadas do elemento. Os argumentos desta função são: o índice que define a posição do elemento no conjunto alvo da seleção e o valor atual das coordendas do elemento. Esta função deverá retornar um objeto contendo as novas propriedades top e left.

O método .offset() para definir coordenadas nos permite alterar a posição de um elemento. A posição do elemento é considerada em relação ao documento. Se a posição do elemento foi definida em folha de estilo com a propriedade position definida para static ela deverá ser sobrescrita para posicionamento com valor relative de modo a possibilitar o novo posicionamento do elemento.

Example:

Definir as coordenadas do segundo parágrafo:

<!DOCTYPE html>
<html>
<head>
  <style>p { margin-left:10px; } </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Alô</p><p>Segundo parágrafo</p>
<script>$("p:last").offset({ top: 10, left: 30 });</script>

</body>
</html>

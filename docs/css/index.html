.css( propertyName ) Retorna: String

Descrição: Retorna o valor de uma propriedade CSS definida para o primeiro elemento do conjunto de elementos alvo do seletor jQuery.

  • criado na versão: 1.0 .css( propertyName )

    propertyNameNome de uma propriedade CSS.

O método .css() fornece um maneira bastante conveniente de se pesquisar o valor de uma determinada propriedade CSS definida para o primeiro elemento do conjunto de elementos alvo de um seletor jQuery. Especialmente se considerarmos as diferentes maneiras como os navegadores acessam aquela propriedades (uso do método getComputedStyle() em navegadores que seguem os Padrões Web versus as propriedades currentStyle e runtimeStyle no Internet Explorer) além da diferença de termos adotados pelos navegadores para nomear certas propriedades CSS. Por exemplo: ao implementar o DOM o navegador Internet Explorer refere-se à propriedade float como styleFloat enquanto os navegadores que seguem os Padrões Web adotam o termo cssFloat. O método .css() leva em consideração estas diferenças e retorna o mesmo resultado independentemente do termo usado pelo navegador. Por exemplo: um elemento que tenha sido flutuado a direita com uso de CSS retornará a string left para cada uma das três linhas de código a seguir:

  1. $('div.left').css('float');
  2. $('div.left').css('cssFloat');
  3. $('div.left').css('styleFloat');

E mais, jQuery interpreta corretamente todas as sintaxes CSS para as propriedades formadas por multi-palavras. Por exemplo: jQuery entende e retorna o valor correto tanto para .css('background-color') como para .css('backgroundColor').

Propriedades CSS abreviadas (por exemplo: margin, background, border) não são suportadas. Por exemplo se você pretende consultar a propriedade margin use: $(elem).css('marginTop') e $(elem).css('marginRight'), e assim por diante.

Exemplo:

Para saber a cor de fundo de um div clicado observe o exemplo.

<!DOCTYPE html>
<html>
<head>
  <style>
div { width:60px; height:60px; margin:5px; float:left; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<span id="result">&nbsp;</span>
<div style="background-color:blue;"></div>
<div style="background-color:rgb(15,99,30);"></div>

<div style="background-color:#123456;"></div>
<div style="background-color:#f11;"></div>
<script>
$("div").click(function () {
  var color = $(this).css("background-color");
  $("#result").html("Este div tem a cor de fundo <span style='color:" +
                     color + ";'>" + color + "</span>.");
});

</script>

</body>
</html>

.css( propertyName, value ) Retorna: jQuery

Descrição: Define uma ou mais propriedades CSS para o conjunto alvo do seletor jQuery.

  • criado na versão: 1.0 .css( propertyName, value )

    propertyName Nome de uma propriedade CSS.

    value Valor definido para a propriedade CSS.

  • criado na versão: 1.4 .css( propertyName, function(index, value) )

    propertyName Nome de uma propriedade CSS.

    function(index, value) Define uma função retornando o valor da propriedade CSS a ser adicionada ao elemento. Os argumentos desta função são: o índice que define a posição do elemento no conjunto alvo da seleção e o valor atual da propriedade.

  • criado na versão: 1.0 .css( map )

    map Um mapa de pares propriedade/valor a ser adicionados aos elementos alvos

Tal como o método .prop() o método .css() torna a atribuição de valorse para propriedades CSS uma tarefa fácil e simples. Este método admite duas sintaxes para seu parâmetro a primeira um par propriedade/valor e a outra um mapa de chaves/valor (conforme a sintaxe para objetos na JavaScript).

Aqui também jQuery interpreta corretamente todas as sintaxes CSS para as propriedades formadas por multi-palavras. Por exemplo: jQuery entende e retorna o valor correto tanto para .css({'background-color': '#ffe', 'border-left': '5px solid #ccc'}) como para .css({backgroundColor: '#ffe', borderLeft: '5px solid #ccc'}). Notar que na sintaxe para DOM as aspas nos nomes das propriedades são opcionais, mas na sintaxe CSS elas são obrigatórias devido ao hífen separador das palavras.

Quando usamos .css() para definir uma propriedade jQuery altera aquelA propriedade de estilização do elemento. Por exemplo: $('#mydiv').css('color', 'green') é equivalente a document.getElementById('mydiv').style.color = 'green'. Definir um valor vazio para uma propriedade — por exemplo: $('#mydiv').css('color', '') — remove a propriedade de stilização do elemento quer ela tenha sido aplicada com declaração de estilo inline, com uso do método .css() da jQuery ou com manipulação do DOM via JavaScript. Contudo não remove um estilo que tenha sido aplicado com uso de uma regra em uma folha de estilos ou via o elemento <style>.

A partir da versão 1.6 da biblioteca jQuery o método .css() aceita valores relativos (ou valores incrementais) tal como ocorre com o método .animate(). Valorese relativos são strings começando com += ou -= destinadas a incrementar um valor. Por exemplo: se o valor de padding-left é 10px, .css( "padding-left", "+=15" ) resulta em um padding-left total igual a 25px.

A partir da versão 1.4 da biblioteca jQuery o método .css() permite que se psse uma função como parâmetro definindo uma propriedade:

$('div.example').css('width', function(index) {
  return index * 50;
});

Neste exemplo a propriedade width dos elementos alvo do seletor jQuery tem seu valor igual ao valor do índice multiplicado por 50.

Exemplos:

Exemplo: O código a seguir altera a cor dos parágrafos para vermelha quando o usuário pass o mouse sobre eles. 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { color:blue; width:200px; font-size:14px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
  <p>Passe o mouse em mim.</p>

  <p>Ou, em mim para ver a mudança de cor.</p>
  
<script>
  $("p").mouseover(function () {
    $(this).css("color","red");
  });
</script>

</body>
</html>

<!DOCTYPE html>

Exemplo: Neste exemplo definimos a cor de todos os parágrafos como vermelha e seu fundo como azul: 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { color:green; }

</head>

  $("p").hover(function () {

Exemplo: Neste exemplo aumentamos o tamanho de um div ao clicá-lo: 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { width: 20px; height: 15px; background-color: #f33; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
  <div>click</div>
  <div>click</div>

<script>
  $("div").click(function() {
    $(this).css({
      width: function(index, value) {
        return parseFloat(value) * 1.2;
      }, 
      height: function(index, value) {
        return parseFloat(value) * 1.2;
      }

    });
  });
</script>

</body>
</html>

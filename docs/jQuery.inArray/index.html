jQuery.inArray()

Categorias: Utilities

jQuery.inArray( value, array ) Retorna: Número

Descrição: Pesquisa por um valor específico na array e retorna o index dele (ou -1 se não for encontrado)

  • versão adicionada: 1.2jQuery.inArray( value, array )

    valueO valor para pesquisa.

    arrayA array que deve ser pesquisada

O método $.inArray() é similar ao método nativo do JavaScript .indexOf() que nele retorna -1 quando não encontra nenhuma ocorrência. Se o primeiro elemento na array for o valor pesquisado, $.inArray() irá retornar 0.

Como JavaScript trata 0 igual a false (ex.: 0 == false, mas 0 !== false), se você estiver verificando a presença do valor na array, deverá verificar se não é igual (ou maior que) -1.

Exemplo:

Retornará o index de alguns elementos na array.

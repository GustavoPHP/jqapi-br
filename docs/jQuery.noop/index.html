jQuery.noop()

Categorias: Utilities

jQuery.noop() Retorna: Function

Descrição: Uma função vazia.

  • versão adicionada: 1.4jQuery.noop()

Você pode usar esta função vazia quando você deseja passar em torno de uma função que não fará nada.

Isso é útil para os autores de plugins que oferecem retornos opcionais, no caso de que nenhum callback for fornecido, pode-se executar o jQuery.noop.

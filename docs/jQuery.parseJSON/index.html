jQuery.parseJSON

Categorias: Utilities

jQuery.parseJSON( json ) Retorna: Object

Descrição: Recebe uma string bem formatada em JSON e retorna um objeto JavaScript

  • adicionado na versão: 1.4.1jQuery.parseJSON( json )

    jsonA string JSON que será transformada.

Passando uma string JSON mal-formatada pode resultar em uma exceção. Por exemplo, algumas strings JSON mal-formatadas:

  • {test: 1} (test não está entre aspas duplas).
  • {'test': 1} ('test' está utilizando aspas simples ao invés de aspas duplas).

Além disso, se você não passar nada, uma string vazia, null ou undefined, 'null' será retornado a partir do parseJSON. Se o navegador fornecer uma implementação nativa de JSON.parse, o jQuery utiliza ele para analisar a cadeia. Para mais detalhes sobre o formato JSON, consulte http://json.org/.

Exemple:

Parse a JSON string.

var obj = jQuery.parseJSON('{"name":"John"}');
alert( obj.name === "John" );

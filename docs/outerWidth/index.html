.outerWidth()

Categories: CSS | Dimensions | Manipulation > Style Properties

.outerWidth( [ includeMargin ] ) Retorna: Integer

Description: Este método retorna o valor da propriedade CSS width definida para o primeiro elemento do conjunto de elementos alvo do seletor jQuery. No valor retornado está incluído o valor de padding, border e opcionalmente o de margin.

  • criado na versão: 1.2.6 .outerWidth( [ includeMargin ] )

    includeMarginValor booleano indicando se deverá ou não ser incluido o valor de margin no cálculo da largura retornada.

Os valores de padding e border top e bottom são sempre incluídos no cálculo de .outerWidth(); se o parâmetro includeMargin estiver presente (true) então margin (left e rigth) também serão incluídas no cálculo.

Este método não se aplica aos objetos window e document, para estes objetos use o método .width().

Exemplo:

Retorna outerWidth do parágrafo.

<!DOCTYPE html>
<html>
<head>
  <style>
  p { margin:10px;padding:5px;border:2px solid #666; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Hello</p><p></p>
<script>var p = $("p:first");
$("p:last").text( "outerWidth:" + p.outerWidth()+ " , outerWidth(true):" + p.outerWidth(true) );

</script>

</body>
</html>

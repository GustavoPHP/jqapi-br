jQuery.isEmptyObject()

Categorias: Utilities

jQuery.isEmptyObject( object ) Retorna: booleano

Descrição: Verifica se o objeto é vazio (não contém propriedades)

  • versão adicionada: 1.4jQuery.isEmptyObject( object )

    objectO objeto que será checado para verificar se ele é vazio.

A partir do jQuery 1.4 este método verifica as propriedades no objeto em si e as propriedades herdadas de protótipos (em que não é utilizado o hasOwnProperty). O argumento deve ser um objeto JavaScript simples; outros tipos de objetos (elementos DOM, string / números), não podem dar resultados consistentes entre os navegadores. Para determinar se um objeto é um objeto JavaScript simples, use $.isPlainObject()

Exemplo:

Verifica o objeto para verificar se ele é vazio.

jQuery.isEmptyObject({}) // true
jQuery.isEmptyObject({ foo: "bar" }) // false

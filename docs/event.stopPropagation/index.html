event.stopPropagation()

Categorias: Events > Event Object

event.stopPropagation()

Descrição: Previne o evento de interferir através da árvore DOM, prevenindo qualquer manipulador pai de notificar o evento.

  • adicionado na versão: 1.0event.stopPropagation()

Nós podemos utilizar event.isPropagationStopped() para determinar se este método foi chamado alguma vez (neste objeto de evento).

Este método funciona para eventos customizados chamado com trigger(), também.

Observe que isso não vai prevenir outros manipuladores no mesmo elemento de rodar.

Notas adicionais:

  • Enquanto o .live() manipula os métodos de eventos uma vez que foi propagado no topo do documento, isso não possibilita parar a propagação de elementos live. Similarmente, eventos chamados por .delegate() sempre irão propagar ao elemento que eles foram delegados; manipuladores de eventos em qualquer elemento abaixo, isso irá ser executado pelo tempo que o manipulador de evento for chamado.

Exemplo:

Mata a interferência no evento de clique.

$("p").click(function(event){
  event.stopPropagation();
  // do something
});

jQuery.isPlainObject()

Categories: Utilities

jQuery.isPlainObject( object ) Returns: Boolean

Descrição: Verifica se o objeto é um objeto simples (criado utilizando "{}" ou new Object).

  • versão adicionada: 1.4jQuery.isPlainObject( object )

    objectO objeto que será checado se é ou não um objeto simples.

Exemplo:

Verifica se o objeto é ou não um objeto simples.

jQuery.isPlainObject({}) // true
jQuery.isPlainObject("test") // false

.outerHeight()

Categories: CSS | Dimensions | Manipulation > Style Properties

.outerHeight( [ includeMargin ] ) Retorna: Integer

Descrição: Este método retorna o valor da propriedade CSS height definida para o primeiro elemento do conjunto de elementos alvo do seletor jQuery. No valor retornado está incluído o valor de padding, border e opcionalmente o de margin.

  • Criado na versão: 1.2.6 .outerHeight( [ includeMargin ] )

    includeMargin Valor booleano indicando se deverá ou não ser incluido o valor de margin no cálculo da altura retornada.

Os valores de padding e border top e bottom são sempre incluídos no cálculo de .outerHeight(); se o parâmetro includeMargin estiver presente (true) então margin (top e bottom) também serão incluídas no cálculo.

Este método não se aplica aos objetos window e document, para estes objetos use o método .height().

Example:

Retorna outerHeight do parágrafo.

<!DOCTYPE html>
<html>
<head>
  <style>p { margin:10px;padding:5px;border:2px solid #666; } </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Hello</p><p></p>
<script>var p = $("p:first");
$("p:last").text( "outerHeight:" + p.outerHeight() + " , outerHeight(true):" + p.outerHeight(true) );</script>

</body>
</html>

.blur()

Categories: Events > Form Events | Forms

.blur( handler(eventObject) ) Retorna: jQuery

Descrição: Atrela um manipulador de evento ao evento JavaScript "blur", ou dispara o evento em um elemento.

  • criado na versão: 1.0 blur( handler(eventObject) )

    handler(eventObject) Função a ser executada toda vez que o evento ocorre.

  • criado na versão: 1.4.3 .blur( [ eventData ], handler(eventObject) )

    eventData Mapa de dados a serem passados ao manipulador do evento.

    handler(eventObject) Função a ser executada toda vez que o evento ocorre.

  • criado na versão: 1.0 .blur()

Nos dois primeiros casos mostrados anteriormente este método é uma forma abreviada para o método .bind('blur', handler) e na terceiro para o método .trigger('blur').

O evento blur ocorre em um elemento quando ele perde o foco. Antigamente este evento aplicava-se somente a elementos de formulário, tal como <input>. Em navegadores modernos o evento se aplica a todos os elementos. Um elemento pode perder o foco via comandos de teclado, tal como tecla Tab ou por clique do mouse em um ponto da página fora do elemento.

Considere a seguinte marcação HTML:

<form>
  <input id="target" type="text" value="Field 1" />
  <input type="text" value="Field 2" />
</form>
<div id="other">
  Dispara do evento
</div>
O disparador do evento pode ser atrelado ao primeiro campo input:
$('#target').blur(function() {
  alert('Chamado o manipulador do evento .blur()');
});

Se o primeiro campo do formulário tem o foco, ao se clicar em um ponto fora dele na página ou se acionar a tecla de tabulação o alerta é mostrado:

Chamado o manipulador do evento .blur()

Para disparar o evento de forma programática chama-se o método .blur() sem argumentos:

$('#other').click(function() {
  $('#target').blur();
});

Agora, ao clicar no disparador o alerta é mostrado.

O evento blur não se propaga pelo efeito bolha (bubble) no Internet Explorer. Assim, scripts que dependem de delegação de evento para blur não funcionam de forma consistente em todos os navegadores. Contudo, a partir da versão jQuery 1.4.2 foi introduzida a alternativa de mapear blur para focusout com seus métodos para delegação de eventos .live() e .delegate().

Para atrelar o evento blur em todos os parágrafos:

$("p").blur();

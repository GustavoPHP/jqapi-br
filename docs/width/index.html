.width() Retorna: Integer

Descrição: Este método retorna o valor da propriedade CSS width definida para o primeiro elemento do conjunto de elementos alvo do seletor jQuery.

  • criado na versão: 1.0 .width()

A diferença entre os métodos .css('width') e .width() é que o segundo retorna um número puro (sem unidade) representando o valor da largura do elemento (por exemplo: 400) e o primeiro retorna o valor da largura com sua respectiva unidade de medida (por exemplo 400px). O método .width() é recomendado quando o valor da altura do elemento destina-se a ser usado em cálculos matemáticos.

Com esse método podemos, também, encontrar a largura da janela (window) do navegador bem como a largura do documento.

$(window).width();   // retorna a largura da janela do navegador (viewport)
$(document).width(); // retorna a largura do documento HTML

Convém notar que .width() retorna a largura correspondente ao conteúdo do elemento de acordo com a propriedade CSS box-sizing (por padrão sem considerar padding, border e margin).

Exemplo:

Este exemplo exemplifica o uso deste método, mostrando a inspeção de três larguras. Atenção que os valores retornados se referem ao iframe que contém o exemplo, por esta razão tais valores são menores dos que aqueles para a página como um todo. Destacamos o elemento body do iframe com um fundo na cor amarela.

<!DOCTYPE html>
<html>
<head>
  <style>
  body { background:yellow; }
  button { font-size:12px; margin:2px; }
  p { width:150px; border:1px red solid; }
  div { color:red; font-weight:bold;  }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="getp">Get Paragraph Width</button>
  <button id="getd">Get Document Width</button>
  <button id="getw">Get Window Width</button>

  <div>&nbsp;</div>
  <p>
    Sample paragraph to test width
  </p>
<script>
    function showWidth(ele, w) {
      $("div").text("The width for the " + ele + 
                    " is " + w + "px.");
    }
    $("#getp").click(function () { 
      showWidth("paragraph", $("p").width()); 
    });
    $("#getd").click(function () { 
      showWidth("document", $(document).width()); 
    });
    $("#getw").click(function () { 
      showWidth("window", $(window).width()); 
    });

</script>

</body>
</html>

.width( value ) Retorna: jQuery

Descrição: Define a propriedades CSS width para o conjunto dos elementos alvo do seletor jQuery.

  • criado na versão: 1.0 .width( value )

    value Um número inteiro definindo a largura em pixels se a unidade de medida não for especificada ou na unidade de medida se especificada.

  • criado na versão: 1.4.1 .width( function(index, width) )

    function(index, width)Define uma função retornando o valor da propriedade CSS width a ser adicionada ao elemento. Os argumentos desta função são: o índice que define a posição do elemento no conjunto alvo da seleção e o valor atual da propriedade width.

Assim, podemos chamar o método .width(value) com o parâmetro value podendo ser uma string (número e unidade de medida) ou simplesmente um número. Se for chamado com somente um número jQuery assume que a unidade de medida é o pixel. Se for chamado com uma string deve-se usar uma unidade de medida CSS válida (tais como 100px, 50%, ou auto). Lembre-se que em navegadores modernos a medida CSS width não inclui padding, border ou margin.

Se não for explicitamente declarada uma medida (como 'em' ou '%') então "px" será concatenado ao valor.

Notar que .width(value) define altura do conteúdo do elemento de acordo com a propriedade CSS box-sizing. Se alterarmos esta propriedade para border-box o método altera outerWidth do box em vez de width do conteúdo.

Exemplo:

Neste exemplo ao se clicar no div alteramos suas larguras para 30px e mudamos sua cor de fundo.

<!DOCTYPE html>
<html>
<head>
  <style>
  div { width:70px; height:50px; float:left; margin:5px;
        background:red; cursor:pointer; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div></div>
  <div>d</div>

  <div>d</div>
  <div>d</div>
  <div>d</div>
<script>

    $("div").one('click', function () {
      $(this).width(30)
             .css({cursor:"auto", "background-color":"blue"});
    });
</script>

</body>
</html>

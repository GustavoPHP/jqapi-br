.height() Retorna: Integer

Descrição: Este método retorna o valor da propriedade CSS height definida para o primeiro elemento do conjunto de elementos alvo do seletor jQuery.

  • criado na versão: 1.0 .height()

A diferença entre os métodos .css('height') e .height() é que o segundo retorna um número puro (sem unidade) representando o valor da altura do elemento (por exemplo: 400) e o primeiro retorna o valor da altura com sua respectiva unidade de medida (por exemplo 400px). O método .height() é recomendado quando o valor da altura do elemento destina-se a ser usado em cálculos matemáticos.

Com esse método podemos, também, encontrar a altura da janela (window) do navegador bem como a altura do documento.

$(window).height();   // retoena a altura da janela do navegador (viewport)
$(document).height(); // retorna a altura do documento HTML

Convém notar que .height() retorna a altura correspondente ao conteúdo do elemento de acordo com a propriedade CSS box-sizing (por padrão sem considerar padding, border e margin).

Exemplo:

Este exemplo exemplifica o uso deste método, mostrando a inspeção de três alturas. Atenção que os valores retornados se referem ao iframe que contém o exemplo, por esta razão tais valores são menores dos que aqueles para a página como um todo. Destacamos o elemento body do iframe com um fundo na cor amarela.

<!DOCTYPE html>
<html>
<head>
  <style>
  body { background:yellow; }
  button { font-size:12px; margin:2px; }
  p { width:150px; border:1px red solid; }
  div { color:red; font-weight:bold; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <button id="getp">Get Paragraph Height</button>
  <button id="getd">Get Document Height</button>
  <button id="getw">Get Window Height</button>

  <div>&nbsp;</div>
  <p>
    Sample paragraph to test height
  </p>
<script>
    function showHeight(ele, h) {
      $("div").text("The height for the " + ele + 
                    " is " + h + "px.");
    }
    $("#getp").click(function () { 
      showHeight("paragraph", $("p").height()); 
    });
    $("#getd").click(function () { 
      showHeight("document", $(document).height()); 
    });
    $("#getw").click(function () { 
      showHeight("window", $(window).height()); 
    });

</script>

</body>
</html>

.height( value ) Retorna: jQuery

Descrição: Define a propriedades CSS height para o conjunto dos elementos alvo do seletor jQuery.

  • criado na versão: 1.0 .height( value )

    value Um número inteiro definindo a altura em pixels se a unidade de medida não for especificada ou na unidade de medida se especificada.

  • criado na versão: 1.4.1 .height( function(index, height) )

    function(index, height)Define uma função retornando o valor da propriedade CSS height a ser adicionada ao elemento. Os argumentos desta função são: o índice que define a posição do elemento no conjunto alvo da seleção e o valor atual da propriedade height.

Assim, podemos chamar o método .height(value) com o parâmetro value podendo ser uma string (número e unidade de medida) ou simplesmente um número. Se for chamado com somente um número jQuery assume que a unidade de medida é o pixel. Se for chamado com uma string deve-se usr uma unidade de medida CSS válida (tais como 100px, 50%, ou auto). Lembre-se que em navegadores modernos a medida CSS height não inclui padding, border ou margin.

Se não for explicitamente declarada uma medida (como 'em' ou '%') então "px" será concatenado ao valor.

Notar que .height(value) define altura do conteúdo do elemento de acordo com a propriedade CSS box-sizing. Se alterarmos esta propriedade para border-box o método altera outerHeight do box em vez de height do conteúdo.

Exemplo:

Neste exemplo ao se clicar no div alteramos suas alturas para 30px e mudamos sua cor de fundo para verde.

<!DOCTYPE html>
<html>
<head>
  <style>div { width:50px; height:70px; float:left; margin:5px;
        background:rgb(255,140,0); cursor:pointer; }  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div></div>
  <div></div>

  <div></div>
  <div></div>
  <div></div>
<script>$("div").one('click', function () {
      $(this).height(30)
             .css({cursor:"auto", backgroundColor:"green"});
    });</script>

</body>
</html>

jQuery.isWindow()

Categorias: Utilities

jQuery.isWindow( obj ) Retorna: boolean

Descrição: Determina se o argumento é uma window.

  • versão adicionada: 1.4.3jQuery.isWindow( obj )

    objObjeto para testar se é ou não uma window.

Isto é utilizado em número de lugares em jQuery para determinar se estamos trabalhando dentro de uma janela de browser (como a window atual ou um iframe).

Example:

Descobre se o parâmetro é uma janela

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  Is 'window' a window? <b></b>
<script>$("b").append( "" + $.isWindow(window) );</script>

</body>
</html>

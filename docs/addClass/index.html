.addClass()

Categories: Attributes | Manipulation > Class Attribute | CSS

.addClass( className ) Retorna: jQuery

Descrição: Este método adiciona uma ou mais classes a cada um dos elementos alvo de um seletor jQuery.

  • criado na versão: 1.0 .addClass( className )

    classNameUm ou mais nomes a serem adicionados ao atributo classe de cada elemento alvo de um seletor jQuery.

  • criado na versão: 1.4 .addClass( function(index, currentClass) )

    function(index, currentClass)Define uma função retornando um ou mais nomes de classe, separados por vírgula, para serem adicionados. Os argumentos desta função são: o índice que define a posição do elemento no conjunto alvo da seleção e o valor da atual classe.

É importante ressaltar que este método não troca o valor da classe atual. Ele simplesmente adiciona o(s) novo(s) valores(s) ao valor que já fora definido para o atributo classe do elemento.

Pode-se adicionar mais de um valor para a classe ao mesmo tempo, separando-os por espaço, conforme mostrado a seguir:

$("p").addClass("myClass noClass");

Este método é usado com frequência em conjunto com o método .removeClass() com a finalidade de proceder uma troca do valor da classe de um elemento conforme mostrado a seguir:

$("p").removeClass("myClass noClass").addClass("yourClass");

Neste exemplo os valores myClass e noClass são removidos de todos os parágrafos e o valor yourClass é adicionado.

A partir da versão 1.4 da biblioteca jQuery o método .addClass() pode receber uma função como argumento.

$("ul li:last").addClass(function() {
  return "item-" + $(this).index();
});

Dada uma lista ordenada com cinco itens - elementos <li> - o código do exemplo mostrado adiciona o valor "item-4" ao atributo classe do último item - elemento <li> - da lista.

Exemplos:

Exemplo: Acrescenta o valor "selected" ao atributo classe dos elementos alvo da seleção jQuery. 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { margin: 8px; font-size:16px; }
  .selected { color:blue; }
  .highlight { background:yellow; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
  <p>Olá</p>
  <p>e</p>
  <p>Adeus</p>
  
<script>
  $("p:last").addClass("selected");
  </script>

</body>
</html>

Exemplo: Passa uma função como argumento do método .addClass() que retorna o valor "green" para ser adicionado ao atributo classe de um div cujo atributo classe já tem o valor "red". 

<!DOCTYPE html>
<html>
<head>
  <style>
  div { background: white; }
  .red { background: red; }
  .red.green { background: green; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
 <div>Este div tem o fundo na cor branca</div>
 <div class="red">Este div tem o fundo na cor verde porque o valor "green" foi adicionado ao seu atributo classe.
   O fundo será vermelho se a função addClass falhar.</div>
 <div>Este div tem o fundo na cor branca</div>
 <p>Não existem divs com fundo na cor verde</p>

<script>
  $("div").addClass(function(index, currentClass) {
    var addedClass;

    if ( currentClass === "red" ) {
      addedClass = "green";
      $("p").text("Existe um div com o fundo na cor verde");
    }
  
    return addedClass;
  });
</script>

</body>
</html>

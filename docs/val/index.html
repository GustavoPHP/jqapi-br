.val() Retorna: String, Array

Descrição: Retorna o valor do primeiro elemento do conjunto de elementos alvo do seletor jQuery.

  • criado na versão: 1.0 .val()

O método .val() destina-se, a princípio, a investigar o valor de elementos de formulários. Para o elemento <select multiple="multiple"> o método .val() retorna um array contendo todos os valores das opções selecionadas.

Para selects e checkboxes pode-se, também, usar os seletores :selected e :checked para inspecionar os valores:

$('select.foo option:selected').val();    // retorna o valor de um select dropdown
$('select.foo').val();                    // forma mais fácil que também retorna o valor de um select dropdown
$('input:checkbox:checked').val();        // retorna o valor de um checkbox marcado
$('input:radio[name=bar]:checked').val(); // retorna o valor de um radio button marcado

Exemplos:

Exemplo: O código a seguir retorna o valor de um select simples e um array de valores de um select múltiplo a seguir mostra os valores retornados. 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { color:red; margin:4px; }
  b { color:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p></p>
  <select id="single">
    <option>Simples</option>
    <option>Simples2</option>
  </select>
  <select id="multiple" multiple="multiple">
    <option selected="selected">Múltiplo</option>
    <option>Múltiplo</option>
    <option selected="selected">Múltiplo</option>
  </select>
<script>
    function displayVals() {
      var singleValues = $("#single").val();
      var multipleValues = $("#multiple").val() || [];
      $("p").html("<b>Simples:</b> " + 
                  singleValues +
                  " <b>Múltiplo:</b> " + 
                  multipleValues.join(", "));
    }

    $("select").change(displayVals);
    displayVals();
</script>
</body>
</html>

Exemplo: Encontrar o valor de um campo input. 

<!DOCTYPE html>
<html>
<head>
  <style>
  p { color:blue; margin:8px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <input type="text" value="some text"/>
  <p></p>
<script>
    $("input").keyup(function () {
      var value = $(this).val();
      $("p").text(value);
    }).keyup();
</script>
</body>
</html>

.val( value ) Retorna: jQuery

Descrição: Define o valor de cada elemento de um conjunto alvo do seletor jQuery.

  • criado na versão: 1.0 .val( value )

    value Uma string de texto ou um array de strings a ser definida como valor de um conjunto de elementos a serem configurados como selecionados ou marcados (selected/checked).

  • criado na versão: 1.4 .val( function(index, value) )

    function(index, value) Função retornando o valor a ser definido.

Este método destina-se, a princípio, a definir um valor para elementos de formulários.

Passando um array de valores permite marcar <input type="checkbox">, <input type="radio"> e <option> em n <select multiple="multiple">. Quando se trata de <input type="radio"> em um grupo de radio button ou de <select multiple="multiple"> as demais opções serão desmarcadas.

O método .val() permite definir um valor passando-o com uma função. A partir da versão jQuery 1.4 a função admite dois parâmetros: índice do elemento e seu valor corrente: 

$('input:text.items').val(function(index, value) {
  return value + ' ' + this.className;
});

O exemplo mostrado incorpora a string " items" ao valor textual dos inputs.

Exemplos:

Example: O código a seguir define o valor de um input. 

<!DOCTYPE html>
<html>
<head>
  <style>
  button { margin:4px; cursor:pointer; }
  input { margin:4px; color:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div>
    <button>Feed</button>
    <button>the</button>
    <button>Input</button>
  </div>
  <input type="text" value="click a button" />
<script>
    $("button").click(function () {
      var text = $(this).text();
      $("input").val(text);
    });
</script>
</body>
</html>

Exemplo: Usar o argumento de uma função para modificar o valor de um input. 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Digite qualquer texto e clique ou tabule para fora do campo.</p>
  <input type="text" value="Digite algo" />
<script>
  $('input').bind('blur', function() {
    $(this).val(function(i, val) {
      return val.toUpperCase();
    });
  });
  </script>
</body>
</html>

<!DOCTYPE html>
<html>

  <style>
  body { color:blue; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>

    <option>Simples2</option>
  </select>

    <option selected="selected">Múltiplo</option>
    <option>Múltiplo2</option>

  <input type="checkbox" name="checkboxname" value="check1"/> check1
  <input type="checkbox" name="checkboxname" value="check2"/> check2
  <input type="radio"  name="r" value="radio1"/> radio1
  <input type="radio"  name="r" value="radio2"/> radio2
<script>
    $("#single").val("Simples2");
    $("#multiple").val(["Múltiplo2", "Múltiplo3"]); 
    $("input").val(["check1","check2", "radio1" ]);

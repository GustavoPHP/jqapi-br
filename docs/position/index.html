.position()

Categories: CSS | Offset | Manipulation > Style Properties

.position() Retorna: Object

Descrição: Este método retorna um objeto contendo os valores das coordenadas (relativas ao elemento-pai) do primeiro elemento do conjunto de elementos alvo do seletor jQuery.

  • criado na versão: 1.2 .position()

O método .position() permite inspecionar a posição de um elemento em relação ao seu elemento-pai. Difere do método .offset() porque com este inspeciona-se a posição de elemento em relação ao documento. Este método é útil para posicionar um elemento próximo a outro quando eles estão contidos em um mesmo elemento no DOM.

O método .position() retorna um objeto contendo as propriedades CSS top e left..

Exemplo:

Inspecionar as coordenadas do segundo parágrafo:

<!DOCTYPE html>
<html>
<head>
  <style>

  div { padding: 15px;}
  p { margin-left:10px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  
<div>
  <p>Hello</p>
</div>
<p></p>

<script>
var p = $("p:first");
var position = p.position();
$("p:last").text( "left: " + position.left + ", top: " + position.top );
</script>

</body>
</html>

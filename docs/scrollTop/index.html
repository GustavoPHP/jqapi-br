.scrollTop() Retorna: Integer

Description: Este método retorna a posição da barra de rolagem vertical do primeiro elemento do conjunto de elementos alvo do seletor jQuery.

  • criado na versão: 1.2.6 .scrollTop()

A posição da barra de rolagem vertical é definida pelo de pixels que mede verticalmente a área fora da visão do usuário. Se a barra de rolagem está posicionada totalmente acima ou se não existe barra de rolagem para o elemento o número retornado por este método é 0.

Exemplo:

Retorna o valor de scrollTop de um parágrafo.

<!DOCTYPE html>
<html>
<head>
  <style>
  p { margin:10px;padding:5px;border:2px solid #666; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>Hello</p><p></p>
<script>var p = $("p:first");
$("p:last").text( "scrollTop:" + p.scrollTop() );

</script>

</body>
</html>

.scrollTop( value ) Retorna: jQuery

Descrição: Define a posição da barra de rolagem vertical para cada um dos elementos alvo do seletor jQuery.

  • criado na versão: 1.2.6 .scrollTop( value )

    value Inteiro definindo a nova posição da barra de rolagem vertical.

O número inteiro define a quantidade de pixels para medir verticalmente a área fora da visão do usuário. Definir posição scrollTop para cada um dos elementos alvo do seletor jQuery.

Exemplo:

Definir scrollTop para um div.

<!DOCTYPE html>
<html>
<head>
  <style>
div.demo {
background:#CCCCCC none repeat scroll 0 0;
border:3px solid #666666;
margin:5px;
padding:5px;
position:relative;
width:200px;
height:100px;
overflow:auto;
}
  p { margin:10px;padding:5px;border:2px solid #666;width:1000px;height:1000px; }
  </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <div class="demo"><h1>lalala</h1><p>Hello</p></div>
<script>$("div.demo").scrollTop(300);
</script>

</body>
</html>
